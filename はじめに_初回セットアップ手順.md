# はじめに — 初回セットアップ手順

Claude Code の応答に連動して、VOICEVOX による音声合成と VMagicMirror の表情・動きの切り替えを行う連携ツールです。
このガイドの通りに進めれば、誰でも 10〜15 分で動かせます。

> **対応環境:** Windows 向けです。Claude Code CLI（ターミナル）、VSCode 拡張の Claude Code、デスクトップ版 Claude の「コード」タブで動作します。「チャット」タブや Web 版 claude.ai には対応していません。

---

## STEP 0: 必要なものを準備する

以下の 5 つを先にインストール・起動しておいてください。

| ソフトウェア | 何に使う？ | ダウンロード |
| --- | --- | --- |
| Python 3.10 以上 | スクリプトの実行に必要 | https://www.python.org/ |
| VMagicMirror v4.x | アバターを画面に表示する | https://baku-dreameater.booth.pm/items/1272298 |
| VOICEVOX | テキストから声を作る | https://voicevox.hiroshiba.jp/ |
| VB-CABLE | 声をアバターの口パクに繋ぐ | https://vb-audio.com/Cable/ |
| Claude Code | AI コーディングアシスタント | https://claude.com/claude-code |

> **VB-CABLE がなくても動きます。** 声は出ますが、アバターの口パク（リップシンク）だけスキップされます。
>
> **Python のインストール時に「Add python.exe to PATH」にチェックを入れてください。** チェックを入れないとセットアップが失敗します。

---

## STEP 1: セットアップスクリプトを実行する

このステップでは、システムの動作に必要な準備を自動で行います。
具体的には以下の 3 つが実行されます:

- **Python 仮想環境の作成** — 他のソフトに影響しない専用の実行環境を作ります
- **必要なパッケージのインストール** — 音声再生やアバター連携に使うライブラリを入れます
- **動作確認** — 上記が正しくセットアップされたかチェックします

このフォルダ内の **`初回セットアップ.bat`** をダブルクリックするだけで OK です。

> **.bat ファイルは安全？** 中身はこのフォルダ内で Python の環境構築をするだけで、ネットワーク通信・レジストリ変更・管理者権限は一切不要です。VSCode やメモ帳で開けば中身を確認できます。
>
> **何回実行しても安全です。** 2 回目以降は既存の環境をスキップします。

最後に `=== セットアップ完了 ===` と表示されれば OK です。

---

## STEP 2: 外部ソフトウェアを起動・設定する

### 2-1. VOICEVOX を起動する

VOICEVOX を開くだけで OK です。バックグラウンドで音声合成サーバーが動きます。

> **Claude Code の使用中は VOICEVOX を起動したままにしてください。** 閉じると音声合成ができなくなります。

### 2-2. VB-CABLE をインストールする（リップシンクが要る場合のみ）

1. VB-CABLE をインストール（PC 再起動が必要な場合あり）

### 2-3. VMagicMirror を起動・設定する

1. VMagicMirror を起動し、好きな VRM モデルを読み込む
2. VMagicMirror Settings → **Hot Key** タブを開く
3. 以下のホットキーを登録し、**「Hot Key を有効化」をオン** にする
4. VMagicMirror Settings → **Word to Motion** タブを開く
5. 各アイテム（1〜10）にブレンドシェイプ（Joy, Angry, Sorrow 等）を設定する
6. VB-CABLE をインストール済みの場合: VMagicMirror Settings → **リップシンク** タブで、マイク入力を **「CABLE Output」** に変更し、**「カメラ非使用時、音声ベースでアバターを動かす」** にチェックを入れる

| ホットキー | 割り当て |
| --- | --- |
| Ctrl+Shift+1 | Word To Motion: 1（笑顔） |
| Ctrl+Shift+2 | Word To Motion: 2（怒り） |
| Ctrl+Shift+3 | Word To Motion: 3（悲しみ） |
| Ctrl+Shift+4 | Word To Motion: 4（驚き） |
| Ctrl+Shift+5 | Word To Motion: 5（真剣） |
| Ctrl+Shift+6 | Word To Motion: 6（照れ） |
| Ctrl+Shift+7 | Word To Motion: 7（困惑） |
| Ctrl+Shift+8 | Word To Motion: 8（冷静） |
| Ctrl+Shift+9 | Word To Motion: 9（喜び） |
| Ctrl+Shift+0 | Word To Motion: 10（普通） |

> **ホットキーは後から設定 GUI で自由に変更できます。**
>
> **注意:**
>
> - 本ツールが送信するのは上記のホットキーのみです。キーボード入力の読み取りや、上記以外のキー送信は一切行いません。
> - 他のアプリケーションと同じホットキーを設定すると、意図しない動作が起きる場合があります。競合しないキーの組み合わせを選んでください。

---

## STEP 3: 設定画面を開く

フォルダ内の **`設定画面を開く.bat`** をダブルクリックしてください。設定画面が開きます。

### 3-1. 仮想ケーブルを設定する（リップシンクが要る場合のみ）

**ボイスタブ** の「仮想ケーブル」で **「CABLE Input」** を選択してください。
これにより、音声が VB-CABLE を通じて VMagicMirror のリップシンクに送られます。

### 3-2. 環境チェックを実行する

画面下部の **「環境チェック」** ボタンを押して **「チェック開始」** をクリックします。

- [OK] VOICEVOX 接続確認
- [OK] 仮想ケーブル（VB-Cable等）検出
- [OK] VMagicMirror 起動確認
- [--] CLAUDE.md ジェネレーター確認

全部 [OK] になれば完璧です。[NG] がある場合は STEP 2 の該当ソフトを確認してください。

### 3-3. 好みの声を選ぶ（任意）

**ボイスタブ** で VOICEVOX のキャラクター・スタイルを選べます。
「この声質で試聴」ボタンで声を確認できます。

### 3-4. キャラクター設定をする（任意）

**スタイルタブ** でアバターの口調を設定できます。
「完成品サンプルから選ぶ」で用意されたキャラを選ぶか、「新規作成」でトーンテンプレートから自分で作成できます。

### 3-5. 保存する

設定が終わったら **「保存」** ボタンを押してください。

---

## STEP 4: CLAUDE.md にルールを追記する

これが一番大事なステップです。この設定をしないと Claude Code がアバターを動かしません。

1. 設定 GUI の **「CLAUDE.md ジェネレーター」タブ** を開く
2. **「生成」** ボタンを押す
3. **「コピー」** ボタンでクリップボードにコピーする
4. `~/.claude/CLAUDE.md` をテキストエディタで開く
5. コピーした内容を **末尾に貼り付けて保存** する

> **`~/.claude/CLAUDE.md` の場所(例):**
> `C:\Users\あなたのユーザー名\.claude\CLAUDE.md`
>
> ファイルが存在しない場合は新規作成してください。
>
> **既に CLAUDE.md に内容がある場合は、既存の内容を消さずに末尾に追記してください。** 既存のルールを削除すると、他の設定が動かなくなる可能性があります。

---

## STEP 5: 動作確認

Claude Code で何か質問してみてください。
応答が終わると：

1. アバターの表情が変わる（ホットキー送信）
2. VOICEVOX で要約文を読み上げる
3. アバターが口パクする（VB-Cable 経由）

全部動いたら **セットアップ完了** です！

---

## うまくいかない時は

| 症状 | 対処 |
| --- | --- |
| 声が出ない | VOICEVOX が起動しているか確認。設定 GUI の「環境チェック」で接続チェック |
| 口パクしない | VB-CABLE がインストールされているか確認。VMagicMirror のリップシンク設定でマイク入力が「CABLE Output」か確認。「カメラ非使用時、音声ベースでアバターを動かす」にチェックが入っているか確認。設定 GUI のボイスタブで仮想ケーブルに「CABLE Input」が選択されているか確認 |
| 表情が変わらない | VMagicMirror の Hot Key が有効か確認。設定 GUI のリアクションタブで「送信」テスト |
| 何も起きない | `~/.claude/CLAUDE.md` にルールが追記されているか確認（STEP 4） |

詳しくは **MANUAL.md** を参照してください。設定 GUI の「ヘルプ」ボタンからも読めます。
